# 🎵 بوت الموسيقى لتليكرام

بوت عربي لتشغيل الموسيقى في المحادثات الصوتية لمجموعات وقنوات تليكرام.

## ✨ المميزات
- تشغيل أغاني من YouTube و Deezer و Saavn
- أزرار تحكم تفاعلية
- دعم قنوات التليكرام
- قوائم تشغيل
- تشغيل 24 ساعة

## 📋 المتطلبات
- Python 3.9+
- FFmpeg
- حساب تليكرام ثانوي (للمساعد)
